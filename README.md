## Getting Started

### Prerequisites

- Node.js
- npm
- Bosta shipment tracking Api

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/0xRamadan/bosta-task.git
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Start the project
   ```sh
   npm run dev
   ```
